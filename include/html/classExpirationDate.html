void ExpirationDate::validate ( std::string  in_value)
private

Validate function.

Validates the ExpirationDate value set during its construction.

It's important to note that the only way year input is invalid is in the case which the in_value has a length lesser than 5 and its digits is not a number, as every possible two number digits is satisfied in the standard(0 to 99). Therefore, the particular year substring value is not required to be check, in contrast to month(01 to 12).
